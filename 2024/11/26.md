＜メモ＞
 
**toggle!**
 
Active Record が提供するメソッド。
 
指定した属性の値を「現在の値とは逆に変更」し、その変更をデータベースに保存する。
 
＜例＞
 
属性が true の場合 → false に変更。
 
属性が false の場合 → true に変更。
 
属性が nil の場合 → true に設定。
 
この変更はデータベースには保存されず、オブジェクトのメモリ上だけで反映される。
